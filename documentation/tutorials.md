# Tutorials

Start with the trading terminal, then learn Swap · Bridge, then finish with Portfolio. Each tab combines an SVG walkthrough with the exact sequence a user should follow.

<TutorialTabs />

## Tutorial order

| Order | Tutorial | What it covers |
|---:|---|---|
| 1 | Terminal | Discover → token terminal → market trade → exact approval → place/cancel/execute order proof |
| 2 | Swap · Bridge | Same-chain swap → fee review → cross-chain route → destination fill |
| 3 | Portfolio | Total value → allocation → chart ranges → assets → recent activity |

Always verify the connected wallet, chain, token addresses, approval amount, minimum received, and explorer receipt before treating an action as complete.
